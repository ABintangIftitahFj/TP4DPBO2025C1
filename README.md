# Desain Program 
Program merupakan implemnetasi Java GUI berbasis swing yang memiliki fungsi CRUD untuk memasukkan data mahasiswa 

### 1 class mahasiswa 
Merupakan kelas yang menampung atribut mahasiswa yaitu 'nim','nama' dan 'jenisKelamin' serta atribut tambahan yaitu 'jalurMasuk' 

### 2 Class jframe menu extend
Berisi GUI serta method insert update delete

# Alur Program
1. **GUI berbentuk tabel untuk melakukan CRUD pada data mahasiswa.**
   
2. **ADD Data**  
   Berguna untuk memasukkan data baru pada form data. User bisa menekan tombol **Add** lalu mengisi atribut yang sudah ada pada form yang tersedia.

3. **UPDATE Data**  
   - Ketika user memilih salah satu data, **mouse listener** akan memilih data yang diklik dan memasukkannya ke dalam form yang tersedia.  
   - User dapat mengubah atribut yang ditampilkan dalam form berbasis **JTextField**, **JComboBox**, serta **JRadioButton** untuk memilih jalur masuk.  
   - Setelah mengedit, user menekan tombol **Update** untuk menyimpan perubahan.

   **Atribut Jalur Masuk:**  
   ![image](https://github.com/user-attachments/assets/3b47cedd-7ac6-46dd-8755-3e179af3a107)


4. **DELETE Data**  
   Berguna untuk menghapus data mahasiswa yang diinginkan.  
   - User memilih data yang ingin dihapus lalu menekan tombol **Delete**.  
   - Sebelum penghapusan, akan muncul **konfirmasi** apakah user ingin menghapus data atau tidak.
